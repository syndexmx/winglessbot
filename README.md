# Wingless pieces Telegram-bot

## Телеграм-бот ассистент для командного разгадывания "Бескрылок"

_______

## Что такое Бескрылка

Стихотворное произведение, часть которого может быть заполнена какой-то известной фразой, строкой из известного произвзведения и так далее.
Эта строка называется "крылом" и ее и надо угадать. Иногда слова крыла по другому разбиваются.

______

Например,
```html
"Умный, што ле?" "Эй, очкастый!"
"Гляньте на никчёмного!"
Так высмеивали часто
[...].
```
```html
Крыло: Неучи учёного
```
